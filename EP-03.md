# What is Python?

-   Python is a **high-level programming language** used to build
    software and solve problems.
-   Created by **Guido van Rossum**.
-   First released in **1991**.
-   Designed to be **simple, readable, and easy to learn**.
-   Python is one of the **most popular programming languages** in the
    world.

# Python Syntax

-   Syntax refers to the **rules for writing code in Python**.
-   Python syntax is designed to be **simple and readable**.

Example:

    print("Hello World")

------------------------------------------------------------------------

# Python Rules

-   Python is **case sensitive**

    name
    Name
    NAME

All are different variables.

-   **Indentation is required** to define code blocks

    if 10 > 5:
        print("10 is greater")

-   **Semicolons are not required**

    a = 10
    b = 20

-   **Variables do not require type declaration**

    x = 10
    name = "Python"

-   **Python uses colon (:) to start a block**

    if x > 5:
        print("x is greater than 5")

------------------------------------------------------------------------

# Comments in Python

-   Comments are used to explain code.
-   Python ignores comments during execution.

Single-line comment:

    # This is a comment
