# Python Variables and Data Types

------------------------------------------------------------------------

# What is a Variable?

-   A **variable** is used to store data in a program.
-   It acts like a **container that holds a value**.
-   The value stored in a variable can be **used and modified later in
    the program**.

Example:

    name = "Python"
    age = 30

------------------------------------------------------------------------

# Why Variables are Used

-   Store information
-   Reuse values in a program
-   Perform calculations
-   Manage program data

------------------------------------------------------------------------

# Creating Variables in Python

-   Variables are created when you **assign a value**.
-   Python does **not require declaring the variable type**.

Example:

    x = 10
    y = 20
    name = "Pintu"

------------------------------------------------------------------------

# Printing Variables

    name = "Python"
    version = 3

    print(name)
    print(version)

Output:

    Python
    3

------------------------------------------------------------------------

# Variable Naming Rules

-   Variable names \*\*must start with a letter or underscore (\_)\*\*.
-   Variable names **cannot start with a number**.
-   Variable names can contain **letters, numbers, and underscores**.
-   Variable names **cannot contain spaces**.
-   Variable names **cannot use Python reserved keywords**.
-   Python variable names are **case-sensitive**.

Examples:

Valid variable names:

    name
    age
    _user
    user1

Invalid variable names:

    1name
    user-name
    user name

------------------------------------------------------------------------

# Good Naming Practices

-   Use meaningful variable names.
-   Use lowercase letters.
-   Separate words using underscores.

Example:

    user_name = "Alex"
    total_marks = 95

------------------------------------------------------------------------

# Assigning Multiple Variables

Python allows assigning values to multiple variables.

Example:

    x, y, z = 10, 20, 30
    print(x)
    print(y)
    print(z)

------------------------------------------------------------------------

# Assigning Same Value to Multiple Variables

    a = b = c = 100
    print(a)
    print(b)
    print(c)

------------------------------------------------------------------------

# What are Data Types?

-   Data types define the **type of data stored in a variable**.

Example:

    x = 10
    name = "Python"

Here:

-   `10` is an **integer**
-   `"Python"` is a **string**

------------------------------------------------------------------------

# Common Python Data Types

Beginner level data types:

-   Integer (`int`)
-   Float (`float`)
-   String (`str`)
-   Boolean (`bool`)

------------------------------------------------------------------------

# Integer (int)

-   Used to store **whole numbers**.

Example:

    age = 25
    year = 2025

------------------------------------------------------------------------

# Float (float)

-   Used to store **decimal numbers**.

Example:

    price = 99.99
    height = 5.8

------------------------------------------------------------------------

# String (str)

-   Used to store **text data**.
-   Strings are written inside **quotes**.

Example:

    name = "Python"
    message = "Hello World"

------------------------------------------------------------------------

# Boolean (bool)

-   Represents **True or False values**.

Example:

    is_student = True
    is_logged_in = False

------------------------------------------------------------------------

# Checking Data Type

Python provides the `type()` function.

Example:

    x = 10
    print(type(x))

Output:

    <class 'int'>

------------------------------------------------------------------------

# Example Program

    name = "Alice"
    age = 25
    height = 5.6
    is_student = True

    print(name)
    print(age)
    print(height)
    print(is_student)

------------------------------------------------------------------------

# Key Points

-   Variables store data.
-   Python automatically determines the data type.
-   No need to declare variable types.
-   Python supports multiple built-in data types.
